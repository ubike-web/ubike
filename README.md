# u-bike

Premium motorbike ride-hailing platform for standard and electric bikes.

## Features
- **Smart Rider Matching**: AI-driven selection of the best available rider.
- **Rider Portal**: Full dashboard for riders with earnings tracking.
- **Secure Wallet**: M-Pesa withdrawal system with PIN verification.
- **Fare Negotiation**: Transparent adjustment system for road and traffic conditions.
- **Google Maps Integration**: Professional route previews and navigation.

## Tech Stack
- **Next.js** (App Router)
- **Firebase** (Auth & Firestore)
- **Genkit** (AI Agent Flows)
- **ShadCN UI** & **Tailwind CSS**

## Getting Started
To run the project locally:
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.
