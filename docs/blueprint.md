# **App Name**: u-bike

## Core Features:

- User Ride Booking & Fare Estimation: Allows users to input pickup and destination, select between 'Normal Bikes' or 'Electric Bikes', and view dynamic, estimated fare prices.
- AI-Driven Smart Matching Tool: An AI tool that intelligently matches available riders with passengers, optimizing for factors like proximity, traffic conditions, and dynamic demand-based pricing.
- Seamless Payment Processing: Integrates secure payment methods for ride fares, with M-Pesa prominently featured as the primary option for users to 'Confirm & Pay'.
- Real-time Ride Tracking: Displays live map tracking of the assigned rider's location, provides an ETA countdown, and updates trip status for the user during their journey.
- Rider Availability & Acceptance Interface: Enables registered riders to toggle their 'Available / Busy' status and manage incoming ride requests (Accept/Decline) via a dedicated dashboard.
- Rider Earnings & Withdrawal Management: Provides riders with a dedicated wallet page to view their current balance, total earnings, and completed trips, including an option to 'Withdraw to M-Pesa'.
- Post-Ride Feedback & Emergency Support: Allows users to rate their ride (5-star system) and leave optional comments, alongside access to in-app emergency and support buttons during a trip.

## Style Guidelines:

- Primary interactive color: Muted Gold (#BF9340). A sophisticated and warm tone for interactive elements, reflecting premium quality.
- Background color: Dark Charred Wood (#2E2B26). A very dark, desaturated, warm-grey hue that provides a premium, deep background analogous to the primary.
- Accent color: Deep Burnt Sienna (#8B2E1E). A richer, more saturated and slightly darker red-orange hue, analogous to the primary, for emphasis and calls to action.
- Headlines and body text: 'Poppins', a geometric sans-serif. Chosen for its clean, modern, and precise feel, suitable for the premium brand style.
- Logo incorporates a white 'u-bike' wordmark, with the 'i' stylized as a motorbike tire, subtly integrating a front-view motorbike concept and a small leaf detail at the end of the bike handle.
- Modern, minimalist line icons are used throughout, such as a subtle lightning bolt (⚡) for electric bikes, maintaining a clean and professional appearance.
- Clean, grid-based layout with generous spacing for an uncluttered and premium feel, establishing a clear visual hierarchy across all sections.
- Hero section features a centered booking interface with tab switches for bike types, and internal booking panels slide left/right akin to a mobile app experience.
- Smooth fade-in and slide-up transitions are used for the hero section elements, with graceful slides and fades for tab switching.
- Subtle micro-interactions (hover, click) on buttons and cards (e.g., rider wallet) and elegant transitions between booking steps, ensuring animations are never excessive.
- An animated 'Searching for rider…' loader provides visual feedback during rider matching.
